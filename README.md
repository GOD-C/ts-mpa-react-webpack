# ts-mpa-react-webpack
支持react + ts语法的多页面打包工具(当然不用react + tsx,同样可以用)


### 已完成功能
- 支持ts语法
- 支持多页面编译打包
- 增加autoprefixer自动添加前缀
- 支持scss语法
- 代码优化分割
- 开启多进程打包
- 美化控制台输出样式
- 使用pug语法
- 支持静态资源文件夹
